FIREBASE STORAGE CORS AYARLARI - ADIM ADIM REHBER
==================================================

SEÇENEK 1: Google Cloud SDK ile (Önerilen)
-------------------------------------------

1. Google Cloud SDK'yı indirin ve kurun:
   https://cloud.google.com/sdk/docs/install

2. Kurulumdan sonra yeni bir terminal açın ve şu komutları çalıştırın:

   gcloud auth login
   (Tarayıcı açılacak, Google hesabınızla giriş yapın)

   gcloud config set project bluecrew-app

   gsutil cors set cors.json gs://bluecrew-app.firebasestorage.app

3. Başarılı olursa "Setting CORS..." mesajını göreceksiniz.

SEÇENEK 2: Firebase Console (Manuel)
-------------------------------------

1. https://console.firebase.google.com/ adresine gidin
2. bluecrew-app projenizi seçin
3. Storage → Settings'e gidin
4. CORS sekmesi varsa JSON'u ekleyin
5. Yoksa Google Cloud Console'a gidin:
   https://console.cloud.google.com/storage/browser?project=bluecrew-app
6. Bucket'ı seçin → Permissions → CORS → Edit
7. cors.json içeriğini ekleyin

NOT: cors.json dosyası proje klasörünüzde hazır!
